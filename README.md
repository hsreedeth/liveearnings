# Live Earnings Timer

A minimalist page that tracks your shift earnings in real-time. 

## Features

- **Real-time earnings calculation** - See your earnings update every second
- **Visual timeline** - Track progress through your shift with an animated progress bar.
- **Pause/Resume** - Take breaks without losing your place
- **Clean, minimal design** - Distraction-free interface thats super minimal.
- **Mobile responsive** - Works oob on desktop and mobile devices
- **No backend required** - Everything runs locally in your browser with localStorage (resets if cookies deleted.)

## Quick Start

**Option 1: Use it directly**
Visit the live demo: [https://hsreedeth.github.io/liveearnings](https://hsreedeth.github.io/liveearnings)

## How to Use

1. **Enter Base Hourly Rate** - Set your hourly wage
2. **Enter Start Time** - When did your shift begin?
3. **Start Shift** - Watch your earnings accumulate in real-time
4. **Pause/Resume** - Control the timer as needed
5. **End & Reset** - Clear everything when your shift is done
